File-parsing API¶

This page contains links to the full documentation for the readers used by xsInterface.

Classes¶

  • DataSettings: User-defined names to be stored on containers.

  • Perturbations: User-defined names and values of the perturbation parameters.

  • SingleSet: Container to collect, store, and process data including:

    • state of the perturbation

    • multi-group macroscopic cross sections

    • multi-group microscopic cross sections

    • kinetic parameters

    • metadata

  • MultiSets: Container to store the data for all the perturbation states as a function of:

    • history,

    • time, and

    • branch
