File-parsing API
================

This page contains links to the full documentation for the readers used
by ``xsInterface``.


Classes
-------

- :ref:`api-datasettings`: User-defined names to be stored on containers.
- :ref:`api-perturbations`: User-defined names and values of the perturbation parameters.
- :ref:`api-singleset`: Container to collect, store, and process data including:
    - state of the perturbation
    - multi-group macroscopic cross sections
    - multi-group microscopic cross sections
    - kinetic parameters
    - metadata
- :ref:`api-multisets`: Container to store the data for all the perturbation states as a function of:
    - history,
    - time, and
    - branch
